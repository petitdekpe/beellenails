SCÉNARIOS DE TESTS COMPLETS (END-TO-END)
═══════════════════════════════════════════════════════════════════════════════

E2E-001: Parcours complet - Réservation de RDV avec paiement
1. Client visite la page d'accueil
2. Clique sur "Prendre RDV"
3. Sélectionne une prestation
4. Choisit une date et un créneau
5. Remplit le formulaire
6. Applique un code promo
7. Choisit un mode de paiement
8. Paie et reçoit la confirmation
9. Reçoit l'email de confirmation
10. Voir le RDV dans son profil

E2E-002: Parcours complet - Achat de formation
1. Client consulte le catalogue formations
2. Clique sur une formation
3. Voit les détails et avis
4. Clique sur "S'inscrire"
5. Paie la formation
6. Accède au premier module
7. Complète tous les modules
8. Obtient son certificat
9. Donne un avis

E2E-003: Parcours admin - Gestion d'un RDV
1. Admin se connecte
2. Voit les nouveaux RDV sur le dashboard
3. Clique sur un RDV en attente
4. Confirme le RDV
5. Email automatique envoyé au client
6. Le jour J, marque le RDV comme terminé

E2E-004: Parcours admin - Création d'une campagne promo
1. Admin crée un code promo
2. Définit -20% pour les nouvelles prestations
3. Valide du 01/01 au 31/01
4. Active le code
5. Client l'utilise lors d'une réservation
6. Admin voit les statistiques d'utilisation
